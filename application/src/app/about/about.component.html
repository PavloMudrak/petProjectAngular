Project Description

Hi, this is my pet project. It is a simple server application without authentication, with only one table Customers (Name, CompanyName, Phone, Email).

The database used is SQL Server.

The server architecture is Layered.

CRUD operations are based on ORM.

I also added a search bar based on a stored procedure, with sorting, searching on all fields, and pagination (also created indexes for this).

The UI is implemented using Angular.

I added pagination navigation buttons.

The ability to select the number of items per page

The ability go to next, previous, first, and last pages.

You can also select the column by which to sort, and sort order.

The total number of pages is displayed.

Fluent validation was used for server-side validation.

Automapper was added to quickly transform objects into others .
